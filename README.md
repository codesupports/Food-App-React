# Food Delivery Practice App - Frontend Developer (React)
This Food Delivery Practice App is designed to simulate the user experience of ordering food from a restaurant via an online delivery platform. The app focuses on building the frontend using React and modern frontend technologies, with a clean, intuitive user interface and responsive design. The primary goal is to provide a seamless and engaging experience for users to browse menus, place orders, and track deliveries. The app is designed with the aim of demonstrating a full-stack development workflow using modern web technologies.

Project Link:<a href="https://codesupports.github.io/Food-App-React/"> https://codesupports.github.io/Food-App-React/</a>

 # Core Features of the App:

<ul>
  <li><b>Home Page:</b> Displays a list of food with images, names, and categories (e.g., Lunch, dinner etc.).</li>
  <li><b>Search Bar:</b> Users can search by food item.</li>
  <li><b>Cart & Checkout:</b> Users can view their cart, modify quantities, and proceed to checkout where they can input delivery details and complete the order.</li>
  <li><b>Order Confirmation:</b> Once the order is placed, users receive success information.</li>
  <li><b>Responsive Design:</b> The app is fully responsive, ensuring a smooth experience across devices (desktop, tablet, and mobile).</li>
</ul>

# Tech Stack:
<h3>Frontend: React</h3>
<ul>
  <li>React or <b>Redux Toolkit</b> for managing global state like the user's cart, selected items etc.</li>
  <li><b>React Router</b>  for navigation between pages (e.g., Home, Cart, and Checkout).</li>
</ul>

<h3>Styling and UI Frameworks: Tailwind CSS</h3>
<ul>
 <li>The user interface is styled using <b>tailwind css</b> for traditional styles for scoped styles in React.</li>
 <li>The design is <b>responsive and mobile-friendly</b>, ensuring usability across devices.</li>
</ul>
